Simple 4-Step Tutorial

Get your AI support widget running in under 10 minutes

1

Download & Setup

Download the files and install Python dependencies. Takes 2 minutes.

2

Customize FAQs

Edit the FAQ file to add your own questions and answers.

3

Start Server

Run one command to start your AI support API server.

4

Embed Widget

Add one line of code to your website and you're done!

Quick Start Commands

# 1. Install dependencies
pip install -r requirements.txt
# 2. Start the server
python main.py
# 3. Add to your HTML
<script src="widget/public/embed.js"></script>

Customize Your Widget

1
Edit FAQ Responses

2
Style & Colors

Change colors, position, and appearance using simple configuration options.

Add AI Power

Configuration Example:
<script src="embed.js" data-widget-config='{
"apiUrl": "http://localhost:8000",
"primaryColor": "#your-brand-color",
"position": "bottom-right",
"greetingMessage": "How can we help?",
"title": "Support Chat"
